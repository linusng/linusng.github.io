Index
  1. Download nvim
    • brew install neovim (macOS)
    • sudo apt-get install neovim (Ubuntu)
  2. Install fisa-vim-config
    • This setup includes all plugin managers, file manager support (NERDTree), python linting and autocomplete.
    • http://vim.fisadev.com/
      • Select Install for NeoVim and follow the steps described
  3. Install my own colorscheme
    • https://github.com/linusng/green-shades.vim
  4. Install CocCommand for state-of-the-art IntelliSense python autocomplete.
    • https://github.com/neoclide/coc.nvim
    • https://github.com/neoclide/coc-python
    • Will have to disable deoplete plugin which was installed along the fisa-vim-config
  5. Install vimwiki
    • https://github.com/vimwiki/vimwiki
    • Follow the guide at https://www.rosipov.com/blog/custom-templates-in-vimwiki/
      • This will teach you how to create and link a custom .html template for VimwikiAll2HTML build.
      • His method does not work, what I did was to append the href to Index.html into the default.tpl file.
      • The default.tpl file is what VimwikiAll2HTML searches for when compiling .wiki to .html files.
      • You can duplicate the original default.tpl to default_original.tpl file just in case you want to revert back.
  6. Install browsersync to track and reflect exported html files by vimwiki
    • https://browsersync.io/#install
  7. Install auto markdown loader
    • Autoload and update .md / .markdown files when opened with nvim
    • https://github.com/shime/vim-livedown
  8. Test
